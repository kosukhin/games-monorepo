Arcade template for Phaser 3

This is a minimal, self-contained arcade-style game skeleton using Phaser 3 with ES modules. It uses a small boot -> preloader -> main scene flow and a simple, extensible structure you can drop into any Phaser project.

How to use:
- Open arcade-template/index.html in a browser (no build step required).
- Or adjust paths and assets to fit your setup.

Project structure:
- arcade-template/index.html
- arcade-template/src/main.js
- arcade-template/src/scenes/BootScene.js
- arcade-template/src/scenes/PreloaderScene.js
- arcade-template/src/scenes/MainScene.js
- arcade-template/README.md

Notes:
- Phaser is loaded via CDN to keep this template lightweight and dependency-free in the repository.
- You can replace the simple placeholder assets with your own graphics and sounds.
